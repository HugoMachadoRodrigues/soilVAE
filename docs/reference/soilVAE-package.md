# soilVAE: Supervised Variational Autoencoder Regression (Keras/TensorFlow via reticulate)

The soilVAE package provides a supervised Variational Autoencoder (VAE)
regression model and tuning utilities for high-dimensional predictors
(e.g., soil spectroscopy).
